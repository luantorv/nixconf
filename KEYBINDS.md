# Keybinds

Keyboard shortcuts used by default. You can change them in `nixconf/home-manager/modules/river.nix`

## General

| Keybind | Action |
|---------|--------|
| Super + Return | Open foot |
| Super + Space | Open/close wofi |
| Super + B | Open/close waybar |
| Super + V | Open/close clipboard |
| Super + L | Open/close energy menu |
| Super + C | Open VSCodium |
| Super + E | Open yazi |
| Super + Shift + E | Open Thunar |
| Super + Alt + E | Exit WM (River) |

## Power Modes

| Keybind | Action |
|---------|--------|
| Super + F1 | Set Power-Save Mode |
| Super + F2 | Set Balanced Mode |
| Super + F3 | Set Performance Mode |

## Windows

| Keybind | Action |
|---------|--------|
| Super + Q | Close focused window |
| Super + D | Fullscreen focused window |
| Super + O | Send focused window to next monitor |
| Super + 0 | Send all windows in focused tag to the next monitor |
| Super + J | Focus next window in the same tag |
| Super + K | Focus previous window in the same tag |
| Super + Shift + J | Swaps positions with the next window |
| Super + Shift + K | Swaps positions with the previous window |
| Super + Period | Main window decreases 5% |
| Super + Comma | Main window increases 5% |

## Navegate for tags

| Keybind | Action |
|---------|--------|
| Super + 1-9 | Focus 1-9 tag |
| Super + Shift + 1-9 | Move focused window to 1-9 tag |

## Screenshoots

| Keybind | Action |
|---------|--------|
| Super + Shift + P | Select rectangular screenshoot |
| Print | Complete monitor screenshoot |

## Wallpapers

| Keybind | Action |
|---------|--------|
| Super + W | Select static wallpaper |
| Super + Alt + W | Select the initial wallpaper for the slideshow (random rotation) |
| Super + Shift + W | Remove wallpaper (default mode) |

## Media

| Keybind | Action |
|---------|--------|
| XF86AudioRaiseVolume | Increase volume 5% |
| XF86AudioLowerVolume | Decrease volume 5% |
| XF86AudioMute | Mute volumes audio output |
| XF86AudioMicMute | Mute volumes audio input |
| XF86AudioPlay | Play/Pause |
| XF86AudioNext | Next |
| XF86AudioPrev | Previous |
| XF86AudioStop | Stop |
